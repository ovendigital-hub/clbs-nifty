CLBS FRAMEWORK

Context → Level → Bias → Strategy · Intraday Nifty Options Decision Engine

All price / VWAP / value-area inputs use Nifty Futures (which has volume) — not spot.

C · Context

Is the environment tradable?

Time of day, India VIX, events, expiry, global cues. A gate — bad context = no trade.

→
L · Level

Where is price vs the map?

VWAP, Initial Balance, prior-day & prior-week Value Area / POC. No level = no edge.

→
B · Bias

Which way is pressure?

Structure, OI/writing, momentum, PCR, max pain + value acceptance → Bullish / Bearish / Neutral.

→
S · Strategy

Does it line up + pay?

Level–Bias confluence, instrument choice, R:R ≥ 1:2 → GO / WAIT / NO-GO.
